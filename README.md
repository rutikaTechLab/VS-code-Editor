## 🧾 Project Overview

**JAS Editor** is an **online code editor** that allows users to write and run **HTML, CSS, and JavaScript code in real-time**.

It provides a **simple, fast, and user-friendly interface** where users can practice frontend development and instantly see output.

## 🎯 Project Objective

The objective of this project is to:

- Provide a **real-time code editing environment**
- Help users **practice frontend development**
- Improve **coding productivity**
- Create a **lightweight and fast editor**
